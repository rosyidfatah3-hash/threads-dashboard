Privacy Policy

Terakhir diperbarui: 7 Juni 2026

1. Data yang Dikumpulkan

Kami mengumpulkan data berikut ketika Anda menggunakan Dashboard Threads API:

  • Access Token: Digunakan untuk mengakses Threads API atas nama pengguna.
  • Keywords: Kata kunci yang digunakan untuk pencarian Threads.
  • Informasi lain yang terkait dengan interaksi pengguna di dashboard.

2. Tujuan Penggunaan Data

Data yang dikumpulkan digunakan untuk:

  • Mengakses Threads API sesuai permintaan pengguna.
  • Menampilkan hasil pencarian kata kunci secara real-time di dashboard.
  • Meningkatkan pengalaman pengguna dan performa dashboard.

3. Durasi Penyimpanan Data

Data disimpan selama aplikasi aktif dan akan dihapus apabila pengguna meminta penghapusan atau token kedaluwarsa.

4. Hak Pengguna

Pengguna berhak untuk:

  • Meminta penghapusan data pribadi kapan saja.
  • Mendapatkan informasi terkait data apa saja yang dikumpulkan.

5. Kontak

Jika ada pertanyaan mengenai Privacy Policy ini atau pengelolaan data pribadi, silakan hubungi: admin@example.com
